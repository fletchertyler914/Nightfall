About Nightfall

Nightfall is a free Bootstrap 3 theme modified by 00:Dark30.
The theme is open source, and you can use it for any purpose, personal or commercial.
Download it here.

Nightfall is a modified version of the Grayscale Bootstrap theme by Startbootstap. You can also download this template from the preview page. The theme is open source, and you can use it for any purpose, personal or commercial.

This theme features stock photos by Gratisography and Pexels Photos. Also along with a custom Google Maps skin courtesy of Snazzy Maps.

Nightfall includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization.
